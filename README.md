README (in Italiano)
Snake – Pygame

Questo progetto è un gioco 2D sviluppato in Python utilizzando la libreria Pygame, ispirato al classico gioco Snake. Il gioco implementa un game loop completo, con gestione degli input da tastiera per il controllo del serpente, movimento continuo su una griglia e aggiornamento in tempo reale dello schermo.
La logica di gioco include la generazione casuale del cibo, il sistema di crescita del serpente a ogni mela raccolta e un contatore di punteggio visualizzato a schermo. Il sistema di collision detection gestisce sia l’impatto con i bordi della finestra sia l’autocollisione del serpente, determinando la fine della partita.
Il campo di gioco è strutturato su una griglia regolare, che rende il movimento fluido e coerente, mentre la grafica minimale e colorata migliora la leggibilità del gameplay. Il progetto è organizzato in modo chiaro, separando la gestione del serpente, del cibo e dello stato di gioco, rendendo il codice facilmente comprensibile, modificabile ed estendibile.

README (in English)
Snake – Pygame

This project is a 2D game developed in Python using the Pygame library, inspired by the classic Snake game. It features a complete game loop, including keyboard input handling for snake movement, continuous grid-based motion, and real-time screen updates.
The game logic includes random food generation, a snake growth mechanic each time an apple is collected, and an on-screen score counter. A collision detection system handles both wall collisions and self-collisions, triggering the game-over condition.
The playfield is based on a regular grid, ensuring smooth and consistent movement, while the minimal and colorful graphics enhance gameplay readability. The project is cleanly structured, with clear separation between snake behavior, food management, and game state logic, making the code easy to understand, maintain, and extend.
